# arithmetica-tui
A command-line wrapper for the arithmetica library.

# Installation
```
curl -s -H "Accept: application/vnd.github.v3.raw" https://api.github.com/repos/avighnac/arithmetica-tui/contents/install.sh | sudo bash
```
